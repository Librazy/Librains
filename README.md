## Librains
官方主页: <http://im.librazy.org/article/179/librains-demo/>  

### 概述
Librains是由Librazy也就是我（目前）独立开发的的开源网页解谜游戏，关卡不定期升级中~
绝对不会真的太没有什么不坑爹的东西
无需Flash支持
开源且自由
支持自动保存进度
采用SHA512和AES加密，您就老老实实猜吧，别老想着破解了
答案目前为全英文，区分大小写，但仅在题目包含了答案的情况下才有大写
